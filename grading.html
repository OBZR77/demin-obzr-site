📊 Как оценивается ОБЗР

Прозрачная система оценивания помогает и ученикам, и родителям понимать, за что выставляется оценка.

ОБЗР — практико-ориентированный предмет, поэтому итоговая оценка складывается из нескольких компонентов, каждый из которых важен.

📚 Теоретические знания (40%)

Оцениваются через тесты, контрольные работы, устные ответы на уроках.

Оценка Критерии Процент правильных ответов
Отлично (5) Полное, глубокое знание материала по всем 11 модулям ОБЗР. Умение применять знания на практике. Грамотная речь, логичные ответы. 90-100%
Хорошо (4) Твёрдое знание материала, но возможны небольшие неточности. Умение применять знания в стандартных ситуациях. 70-89%
Удовлетворительно (3) Знание основных понятий и правил, но есть пробелы. Затруднения при применении знаний на практике. 50-69%
Неудовлетворительно (2) Значительные пробелы в знаниях. Неспособность применить знания даже в простых ситуациях. Менее 50%

🎖️ Практические навыки (30%)

Оценивается выполнение нормативов: строевая подготовка, оказание первой помощи, действия в ЧС, физическая подготовка.

Оценка Критерии
Отлично (5) Строевая подготовка: Уверенное выполнение команд, правильная стойка, повороты, движение строем.
Первая помощь: Быстрое и правильное наложение повязок, жгута, проведение СЛР.
Действия в ЧС: Чёткое следование алгоритму, отсутствие ошибок.
Хорошо (4) Выполнение нормативов с незначительными ошибками, которые не влияют на конечный результат. Небольшие задержки при выполнении.
Удовлетворительно (3) Выполнение нормативов с существенными ошибками, но с пониманием общей логики действий. Медленное выполнение.
Неудовлетворительно (2) Неспособность выполнить норматив даже после повторного показа. Грубые ошибки, представляющие опасность.

🏕️ Учебные сборы (20%)

Оцениваются участие, дисциплина, выполнение программы сборов (5 дней для 10 класса, 3 дня для 8 класса).

Оценка Критерии
Отлично (5) ✓ Полное присутствие на всех занятиях
✓ Образцовая дисциплина и форма одежды
✓ Активное участие во всех мероприятиях
✓ Помощь товарищам
✓ Выполнение всех нормативов на "отлично"
Хорошо (4) ✓ Присутствие на всех занятиях
✓ Соблюдение дисциплины
✓ Выполнение программы с незначительными замечаниями
✓ Выполнение большинства нормативов на "хорошо"
Удовлетворительно (3) ✓ Пропуски по уважительной причине (с медицинской справкой)
✓ Замечания по дисциплине
✓ Выполнение программы с существенными замечаниями
✓ Выполнение нормативов на "удовлетворительно"
Неудовлетворительно (2) ✗ Неявка на сборы без уважительной причины
✗ Грубые нарушения дисциплины
✗ Невыполнение программы
✗ Отказ от выполнения нормативов
⚠️ ВАЖНО: Учебные сборы обязательны для всех учащихся (кроме девочек 8 класса по желанию). Неявка без уважительной причины приравнивается к неуспеваемости по предмету.

🏆 Проектная работа (10%)

Исследовательские работы, творческие задания, участие в олимпиадах и конкурсах по ОБЗР.

Оценка Критерии
Отлично (5) Проект демонстрирует глубокое понимание темы, оригинальность подхода, практическую значимость. Качественное оформление, уверенная защита. Участие в олимпиадах муниципального/регионального уровня.
Хорошо (4) Проект выполнен качественно, но без оригинальности. Есть незначительные недочёты в оформлении или защите. Участие в школьных олимпиадах.
Удовлетворительно (3) Проект выполнен формально, без глубокого погружения в тему. Существенные недочёты в оформлении. Слабая защита.
Неудовлетворительно (2) Проект не выполнен или выполнен с грубыми нарушениями требований. Плагиат.

Примеры проектов: Разработка плана эвакуации для своего дома, создание обучающего видео по первой помощи, исследование истории воинской части в родном городе.

📝 Примеры заданий

Пример теста (модуль № 8 "Первая помощь")
Вопрос: В какой последовательности выполняются действия при сердечно-лёгочной реанимации (СЛР)?
А) 30 нажатий на грудину → 2 искусственных вдоха → повтор
Б) 15 нажатий → 2 вдоха → повтор
В) 2 вдоха → 30 нажатий → повтор
Г) Непрерывные нажатия без вдохов
Правильный ответ: А) 30:2 — это стандартный протокол СЛР для взрослых и детей старше 8 лет.
Пример практического норматива (строевая подготовка)
Задание: Выполнить повороты на месте (направо, налево, кругом) по команде преподавателя.
Критерии оценки "отлично":
✓ Правильная стойка перед поворотом
✓ Чёткое выполнение поворота на 90° (180° для "кругом")
✓ Отсутствие лишних движений
✓ Фиксация положения после поворота
✓ Выполнение за 1-2 секунды
Пример проектного задания
Тема: "Разработка плана действий семьи при пожаре в квартире"
Требования: План эвакуации (схема квартиры), список действий для каждого члена семьи, определение места сбора, список необходимых документов/вещей для эвакуации.
Критерии оценки "отлично":
✓ Подробная схема с указанием всех выходов
✓ Реалистичный и проработанный план действий
✓ Учёт особенностей членов семьи (дети, пожилые)
✓ Качественное оформление
✓ Практическая отработка плана с семьёй

❓ Частые вопросы об оценивании

Можно ли пересдать контрольную работу, если получил "неудовлетворительно"?
Да, можно. Пересдача проводится во внеурочное время по согласованию с преподавателем. Рекомендуется сначала разобрать ошибки и подготовиться повторно.
Учитывается ли посещаемость при выставлении оценки?
Посещаемость влияет косвенно: пропуски приводят к пробелам в знаниях и невозможности выполнить практические задания. Пропуски по уважительной причине (болезнь) не влияют на оценку, но материал нужно изучить самостоятельно.
Обязательно ли участвовать в проектной работе?
Проектная работа составляет 10% итоговой оценки, поэтому настоятельно рекомендуется. Однако если ученик отлично справляется с теорией, практикой и сборами, отсутствие проекта не критично.
Что делать, если ребёнок не может участвовать в учебных сборах по медицинским показаниям?
Необходимо предоставить медицинскую справку. В этом случае оценка за сборы (20%) заменяется на альтернативное задание: реферат, презентация или тест по программе сборов. Детали обсуждаются индивидуально.
Влияет ли дисциплина на уроке на оценку?
Дисциплина — важная часть ОБЗР, особенно для модуля № 2 "Военная подготовка". Систематические нарушения дисциплины могут снизить оценку за поведение, но не влияют напрямую на оценку за знания, если ученик справляется с программой.

⚠️ Важные моменты

  • Итоговая оценка выставляется за полугодие и год с учётом всех компонентов (теория, практика, сборы, проекты).
  • Учебные сборы обязательны. Неявка без уважительной причины приравнивается к академической задолженности.
  • Практические навыки невозможно освоить дистанционно. Присутствие на уроках критически важно.
  • Родители могут в любой момент уточнить текущую успеваемость через электронный дневник или лично у преподавателя.
  • При возникновении сложностей с освоением материала рекомендуется сразу обратиться к преподавателю для индивидуальной консультации.
